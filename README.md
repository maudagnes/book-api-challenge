# Book API Challenge

REST API for storing and retrieving books using Play Framework (Java).

## Technologies
- Play Framework (Java)
- H2 Database
- JUnit 5

## Setup Instructions

### Prerequisites
- Java 21
- SBT 1.9+

### Running the Application

1. Clone the repository
2. Run `sbt run`
3. API available at `http://localhost:9000`

### Running Tests
sbt test

## API Endpoints

### POST /books
Create new books

### GET /books
Retrieve all books
